# Tokenization Techniques for LLM
This project implements tokenization methods used in Large Language Models.
## Implemented Methods
- Byte Pair Encoding (BPE)
- WordPiece
- Basic whitespace tokenizer
## Why Tokenization Matters
Tokenization converts raw text into numerical tokens for LLM training.

```bash
pip install -r requirements.txt
python tokenizer.py
